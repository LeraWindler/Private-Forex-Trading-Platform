# FHEVM Example Documentation

Welcome to the FHEVM example documentation. This project demonstrates how to build privacy-preserving applications using Fully Homomorphic Encryption.

## What is FHEVM?

FHEVM (Fully Homomorphic Encryption Virtual Machine) enables smart contracts to perform computations on encrypted data without decrypting it. This allows building truly private applications on the blockchain.
